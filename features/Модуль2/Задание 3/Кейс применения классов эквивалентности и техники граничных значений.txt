Минимальное количество тест кейсов будет равно 5:
1. Скидка 0% - значение до нижней границы, любое значение переменной от 0 до 999
2. Скидка 5% - граничное значение, значение переменной 1000
3. Скидка 5% - значение между границ, любое значение от 1001 до 1999
4. Скидка 10% - граничное значение, значение переменной 2000
5. Скидка 10% - значение после верхней границы, любое значение переменной от 2001
